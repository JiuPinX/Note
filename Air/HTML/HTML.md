搞这玩意，方便理解，功在当代，利在千秋！

## 示例

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Good morning!</title>
</head>
<body>
    <h1>Hello world!</h1>
</body>
</html>
```
