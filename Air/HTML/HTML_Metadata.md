HTML Metadata 是 HTML 的一个模块，含有页面的相关信息，包括样式、脚本及数据，能帮助一些软件（例如搜索引擎、浏览器 等等）更好地运用和渲染页面。对于样式和脚本的元数据，可以直接在网页里定义，也可以链接到包含相关信息的外部文件。

## 示例

```
<head>
    ......(HTML Metadata)
</head>
```
